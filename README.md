# 🧩 Codeit Bootcamp Missions

> A collection of Codeit bootcamp missions and practice projects.

코드잇 부트캠프에서 수행한 미션들을 정리한 저장소입니다.  
데이터 분석 · 머신러닝 · 딥러닝 · 컴퓨터 비전 · NLP · MLOps까지 단계적으로 경험한 기록입니다.

---

## 📚 Mission List

각 미션별 상세 내용은 `mission01` ~ `mission18` 디렉토리에서 확인할 수 있습니다.

1. **파이썬 응용하기**  
   - Python 기본 문법을 응용한 미니 미션 모음

2. **Hotel Booking Demand Datasets 분석**  
   - 호텔 예약 취소 여부 분석 및 인사이트 도출

3. **자전거 대여 시스템 데이터 분석**  
   - 자전거 대여 수요 분석 및 예측 실습

4. **포르투갈 은행의 마케팅 데이터 분석**  
   - 마케팅 캠페인 효과 분석 및 예측 모델링

5. **손상된 문서를 복원**  
   - 손상된 이미지(문서)를 복원하는 딥러닝 AutoEncoder 실습

6. **흉부 X-Ray 사진으로 폐렴 환자 분류**  
   - CNN 기반 폐렴 진단 이미지 분류 모델 구현

7. **개·고양이 얼굴 Object Detection (SSD)**  
   - SSD 모델을 활용한 얼굴 영역 탐지(Object Detection)

8. **축구 경기 영상 Semantic Segmentation (U-Net)**  
   - U-Net으로 골대, 심판, 선수, 관중 등 객체를 픽셀 단위로 분할

9. **cGAN으로 FashionMNIST 이미지 생성**  
   - 조건부 생성 모델(cGAN)을 이용한 패션 아이템 이미지 생성

10. **뉴스 카테고리 분류 (텍스트 분류)**  
    - Word2Vec / FastText / GloVe 임베딩 + RNN/LSTM/GRU 기반 분류 모델

11. **한국어 → 영어 기계 번역 (Seq2Seq + Attention)**  
    - Seq2Seq 기본 모델 및 Attention 적용 모델 구현

12. **문서 요약 모델 구현 (Transformers)**  
    - Hugging Face Transformers 기반 추상적/추출적 요약 실습

13. **쇼핑몰 리뷰 감성 분석 (Full FT / PEFT)**  
    - 전체 파인튜닝 / 파라미터 효율적 파인튜닝 비교 실험

14. **LangChain 기반 RAG 시스템 구현**  
    - 문서 기반 질의응답(RAG) 파이프라인 설계 및 실험

15. **도커 기반 워크플로우 설계 미션**  
    - 협업 시나리오 기반 Dockerfile 작성 및 컨테이너 워크플로우 구성

16. **모델 양자화 / ONNX 변환**  
    - 모델 경량화 및 서빙을 위한 양자화 & ONNX 실습

17. **손글씨 숫자 인식 Web App (Streamlit)**  
    - 브라우저에서 직접 숫자를 그리면 예측해 주는 Streamlit 서비스

18. **영화 리뷰 감성 분석 웹 애플리케이션**  
    - 영화 정보 + 리뷰 + 감성 분석 결과를 보여주는 웹 서비스
