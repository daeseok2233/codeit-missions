# codeit-missions
A collection of Codeit bootcamp missions.

1. 파이썬 응용하기
2. Hotel Booking Demand Datasets 분석
3. 자전거 대여 시스템 데이터 분석
4. 포르투갈 은행의 마케팅 데이터 분석
5. 손상된 문서를 복원
6. 흉부 X-Ray 사진을 바탕으로 폐렴 환자를 구분
7.  SSD 모델을 활용하여 개와 고양이의 얼굴(Face) 영역을 감지하는 Object Detection 작업을 수행
8. U-Net을 이용해 축구 경기 영상 내의 다양한 객체(예: 골대, 심판, 선수, 관중 등)를 픽셀 단위로 분할하는 Semantic Segmentation 작업
9. cGAN 모델을 활용하여 FashionMNIST 데이터셋의 각 패션 아이템(예: 티셔츠, 바지, 스니커즈 등)을 조건부로 생성하는 작업을 수행
10. 텍스트 데이터를 입력으로 받아 뉴스의 카테고리를 예측하는 딥 러닝 모델을 구현(Word2Vec, FastText, GloVe) (LSTM, GRU, RNN)
11. 기계 번역 실습으로, 한국어 문장을 영어로 번역하는 모델을 구축하는 프로젝트(Seq2Seq 기본 모델, Attention 적용 모델)
12. Hugging Face transformers 라이브러리를 사용하여 문서 요약 모델을 구현
13. 쇼핑몰 리뷰 데이터를 대상으로 감성 분석 ( Full Fine-Tuning / PEFT )
14. LangChain을 이용해 RAG 시스템을 구현한 뒤, 사용된 문서와 관련된 질문을 하고 적절한 답변이 나오는지 확인
15. 두 명의 연구자가 협업하는 아래 시나리오를 참고하여 도커 기반 워크플로우를 설계하고, 필요한 도커파일을 작성하는 미션
16. 모델 양자화 / ONNX
17. 사용자가 웹 인터페이스에서 손으로 숫자를 그려 입력하면 해당 숫자를 예측하는 Streamlit 기반 웹 서비스를 개발
18. 영화 정보, 사용자 리뷰, 리뷰 감성 분석을 표시하는 웹 애플리케이션을 개발
